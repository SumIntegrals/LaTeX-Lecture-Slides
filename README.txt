READ ME
(Updated 2026-05-12)
This is the LaTeX source code for my lecture notes derived from miscellaneous sources for the various courses I teach.
Some of the earlier versions included images (whose sources are not entirely known).
Where possible, I have created the figures and images myself using tikZ, pgfPlot, or other code within the document.

Development has spanned several years, and older versions are likely lost or too riddled with errors that newer versions are preferable.
The goal of these lecture notes is to provide a clear and comprehensive explanation for the various topics so students may access the information reliably.

While no set of notes is perfect, I regularly update these notes as I discover typos, unclear wording, or general issues.
The notes will also be updated as better examples, explanations, or organization of topics occurs.
There likely will be empty folders and blank sections for long periods of time before I eventually get around to writing those sections.

The main courses I am prioritizing the development of are:
- Single-Variable Calculus (commonly called "Calculus 1 and 2")
- Multivariable Calculus (commonly called "Calculus 3")
- Discrete Mathematics (with "An Introduction to Higher Mathematics")
- Axiomatic Geometry
- Precalculus

Other courses I would like to fully develop eventually are:
- Linear Algebra
- Differential Equations
- Algebra, Geometry, and Trigonometry
- Introduction to Topology
- Real Analysis
- Complex Analysis
- Differential Geometry
- Algebraic Geometry
- Differential Topology
- Algebraic Topology
- Vector Calculus (or "Advanced Calculus")
- Mathematical Logic
- and so much more...

The files should be organized reasonably by "TopicName" for newer builds; however, in older builds I had been using "SecX-Y" notation to adhere to book numbering.

General compiling should be doable with modern LaTeX compilers, but certain oddities still seem to appear.
My coloring schemes have taken a backseat lately to more "professional" looking aesthetics.
All of the documents will have color schemes at the beginning using various labels such as \bgcolor for background color, etc.

This is fundamentally mathematics, so copyright information is tenuous.
However, some of the topic organization follows conventions used by most authors - not any one specific text.
I have regularly added topics to sections, moved topics earlier or later, and ignored topics which are commonly left as optional in my lectures.
In short, these notes reflect my actual preferences, though I will usually relegate any optional discussions to the end of sections or label optional sections with asterisks.
